# Setac² — Semana Acadêmica UTFPR 2025

> Site oficial da **Setac²** (Semana Acadêmica de Tecnologia e Computação), evento organizado pelo CACIC-SH — Centro Acadêmico de Ciências da Computação da UTFPR Campus Santa Helena.

🔗 **Acesse em produção:** [setac2-2025.vercel.app](https://setac2-2025.vercel.app)

---

## 📌 Sobre o Projeto

A Setac² é a maior semana acadêmica de tecnologia da UTFPR Campus Santa Helena. Este projeto é o site do evento da edição de 2025, com programação completa, informações sobre mini cursos, palestras e a competição de programação.

O site apresenta:

- 📅 **Programação** — agenda completa dos três dias do evento (11, 12 e 13 de novembro de 2025)
- 🎓 **Mini Cursos & Palestras** — perfis dos palestrantes e horários
- 💻 **Competição de Programação** — regras, cronograma, critérios de pontuação e premiação
- 📲 **Links de inscrição** e redes sociais do CACIC-SH

## 🚀 Como Rodar Localmente

### Pré-requisitos

- [Node.js](https://nodejs.org/) 18+
- [pnpm](https://pnpm.io/) (recomendado) ou npm

### Instalação

```bash
# Clone o repositório
git clone https://github.com/GUSTAVOHOOO/Setac2-2025.git

# Acesse a pasta do projeto
cd Setac2-2025

# Instale as dependências
pnpm install
# ou
npm install
```

### Executando em desenvolvimento

```bash
pnpm dev
# ou
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

### Build para produção

```bash
pnpm build
pnpm start
```

## 🏆 Sobre a Competição de Programação

A competição de programação foi realizada em **13/11/2025** das 13:30 às 17:00.

- **Formato:** Solo ou equipe (até 3 pessoas)
- **Linguagens aceitas:** C, C++, Java, Python, JavaScript
- **Ranking:** por número de problemas resolvidos + tempo total
- **Premiação:** Medalha de ouro, prata e bronze + certificados
